## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/naye-pankh-foundation.git
```

Move to the project

```bash
cd naye-pankh-foundation
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
